# Zara

## 프로젝트 기간 
- 2021/06~


## 프로젝트 개요
- 자전거를 좋아하고 자전거에 대해 관심있는 사람들의 커뮤니티 사이트 , 중고 자전거 거래도 가능 

## 기술 스택
- 사용언어 : Java(1.8버전) 
- 서버 : Apache Tomcat9
- 프론트 엔드 : 부트스트랩 , Jquery ,JavaScript , JSP 
- 백엔드 : Spring , AJax
- 협업 및 툴 : Github, ErdCloud , Eclipse , SqlDeveloper
- 데이터베이스 : Oracle,Mybatis
